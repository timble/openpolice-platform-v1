Streaming lokale inhoud

Met AllVideos Reloaded, is het voor iedereen mogelijk om media te streamen wat is geüpload middels de standaard Joomla! methode of een dedicated media manager. Deze optie kan behulpzaam zijn voor een aantal redenen. Bijvoorbeeld, stel je voor een bedrijfswebsite moet bedrijfsvideos afbeelden zonder de "YouTube" logo's! Een andere denkbare situatie is de regelmatige downtime van sommige aanbieders waardoor je een lege pagina krijgt in je website.

Ondersteunde bestandsformaten

Je kunt bijna ieder type video formaat uploaden op je Joomla! website welke vandaag de dag gebruikt word op het internet. Van Windows Media tot QuickTime, Flash, DivX of Real Media en natuurlijk MP3!

Hoe kun je video (of audio) bestanden toevoegen binnen je inhoud welke je hebt geüploaded, gebruik makend van de AllVideos Reloaded Plugin? Simpel!

Voordat je begint, dien je de hoofd video- en audio- mappen te maken. Als je standaarden aanhoud van de plugin zijn parameters, zullen deze mappen als gevolgd moeten zijn videos respectievelijk audio en dienen deze te staan in de lokale site's bestanden directory. (Met Joomla! 1.5 is deze directory te configureren in de Tab "Systeem Instellingen" van de Globale Configuratie.) Als je de standaard vooringestelde instellingen gebruikt, kom je uit in de ROOT/images/stories/videos respectievelijk ROOT/images/stories/audio directories. Na het uploaden van de video's holiday.flv en party.avi in de video map, kun je de video frames in de inhoud creëren, door simpel gebruik te maken van {flv}holiday{/flv} respectievelijk {avi}party{/avi}. Let op: Het bestandspad is automatisch ingesteld en de bestandsextensie (altijd kleine letters!) word automatisch toegevoegd door de plugin. Als je de video mappen vult met je videos, wil je deze misschien wel organiseren middels submappen, indien - bijvoorbeeld - je een bestand hebt genaamd ROOT/images/stories/videos/holiday2007/film1.flv, zou de corresponderende tekst zijn: {flv}holiday2007/film1{/flv}. Begrijp je het? Ok, vanaf nu zullen we refereren aan de tekst string als gebruikt tag in deze documentatie. De volgende tabel laat de ondersteunende bestandsformaten zien voor lokale streaming en de corresponderende tags die gebruikt moeten worden:

Formaat Tag te gebruiken Speler die word gebruikt Opmerkingen
Flash video {flv}name{/flv} JW Media Player Het meest aanbevolen formaat voor multimedia web inhoud. Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .flv) is automatisch toegevoegd.
Shockwave flash video {swf}name{/swf} JW Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .swf) is automatisch toegevoegd.
Flash animation {flash}name{/flash} - This tag can be used to embed native and interactive flash animations (.swf) which do not work with the {swf} tag. The path is prepended automatically and the file extension (lowercase .swf) is appended automatically.
MP3 Audio {mp3}name{/mp3} JW Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .mp3) is automatisch toegevoegd. De grootte van de speler is gereduceerd tot alleen het controle paneel. In order to play properly, the MP3 media must be encoded with constant bitrate (CBR) and using a sample frequency of 11, 22, 33 or 44 kHz. This limitation is imposed by using Flash in general, not a limitation of JW Media Player!
3GP mobile media {3gp}name{/3gp} JW Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .3gp) is automatisch toegevoegd.
RBS Audio {rbs}name{/rbs} JW Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .rbs) is automatisch toegevoegd.
Ieder van de bovenstaande {auto}name.flv{/name} JW Media Player Automatisch bestandstype herkend door JW Media Player. Het pad is automatisch ingesteld, de bestandsextendsie dient in de tag gebruikt te worden.
Mpeg4 H.264 {m4v}name{/m4v} JW Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .m4v) is automatisch toegevoegd.
Windows Media Video {wmv}name{/wmv} Windows Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .wmv) is automatisch toegevoegd. Door het feit dat er een groot aantal verschillende codecs beschikbaar zijn, kun je nooit voorspellen of een bepaald bestand afspeelbaar is op een bepaalde client.
Windows Media Audio {wma}name{/wma} Windows Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .wma) is automatisch toegevoegd.
AVI {avi}name{/avi} Windows Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .avi) is automatisch toegevoegd. Door het feit dat er een groot aantal verschillende codecs beschikbaar zijn, kun je nooit voorspellen of een bepaald bestand afspeelbaar is op een bepaalde client.
MPEG {mpg}name{/mpg} Windows Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .mpg) is automatisch toegevoegd.
MPEG {mpeg}name{/mpeg} Windows Media Player Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .mpeg) is automatisch toegevoegd.
QuickTime {mov}name{/mov} Apple Quicktime Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .mov) is automatisch toegevoegd.
MP4 {mp4}name{/mp4} Apple Quicktime Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .mp4) is automatisch toegevoegd.
Real Media {rm}name{/rm} RealPlayer Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .rm) is automatisch toegevoegd.
Real Media {ram}name{/ram} RealPlayer Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .ram) is automatisch toegevoegd.
DivX {divx}name{/divx} DivX webPlayer Het pad is automatisch ingesteld en de bestandsextensie (kleine letters .divx) is automatisch toegevoegd.

Voor alle hierboven getoonde tags, er bestaat een tweede variant welke niet automatisch het pad insteld en niet de bestandsextensie automatisch toevoegd. (Met andere woorden: gebruiken van de naam zoals deze is). Deze variant kan gebruikt worden voor zowel lokale streaming als voor verafgelegen media. Je kunt deze variant gebruiken door het woord remote toe te voegen aan de originele tekst. Voorbeeld: {flvremote}http://www.yoursite.com/images/stories/videos/name.flv{/flvremote}
