Welkom bij AllVideos Reloaded!

Snelstart

Voordat je lokale media invoegd, gebruik eerst Joomla's media manager om twee folders aan te maken genaamd audio en videos in je images/stories map. Gebruik daarna Joomla's media manager om je media daar te uploaden. Indien dit niet goed werkt, dien je mogelijk Joomla's globale instellingen te wijzigen om het uploaden van andere bestanden dan alleen afbeeldingen toe te staan.

Tijdens het wijzigen van je content, gebruik de nieuwe AVR Media knop aan de onderzijde van het editor scherm om media in te voegen. Voor embedded Verafgelegen videos van een video sharing site, in de dialoog die opent, simpel plak de URL van de pagina die de video afbeeld in het eerste veld, klik daarna op RETURN om de automatische herkenning te starten. Indien de URL word ondersteund, zal de rest van het formulier automatisch worden ingevuld. Het laatste veld laat de media tag zien, die gewijzigd kan worden om attributen toe te voegen. Als laatste, klik de Invoegen knop en de content van dit veld zal worden ingevoegd op de huidige cursor positie in de editor.

Om lokale media in te voegen, gebruik dezelfde AVR Media knop, maar selecteer de Lokale Media tab. Daar kun je een van de eerdere geuploade media bestanden selecteren. Na de selectie van een media file, zullen de rest van de velden automatisch worden ingevuld.

Voor meer informatie - met name voor je gebruik gaat maken van de geavanceerde mogelijkheden van het component bekijk -, a.u.b. lees de bijgevoegde documentatie. AllVideos Reloaded tracht het zo makkelijk als mogelijk voor je te maken, embedden van multi media is echter een complexe materie. Om het beste uit je media te kunnen halen, is het lezen van de documentatie een vereiste. Je kunt de documentatie op ieder moment aanroepen in de plugin omschrijving van de Content - AllVideos Reloaded plugin of door te klikken op de Help knop in een van de AllVideos Reloaded component adviezen.
De Demo site verstrekt eveneens een collectie van handige voorbeelden.

Indien je hulp nodig hebt, kun je dat krijgen in deze discussie draad op de internationale Joomla forums.
Indien je je favoriete video sharing aanbieder mist of je hebt een idee over een nieuwe mogelijkheden, voeg dan een aanvraag toe op de Project Tracker.
