Bienvenido a AllVideos Reloaded!

Guía Rápida

Antes de insertar videos/audios locales, debe usar el administrador de medios de Joomla para crear dos directorios llamdos audios y videos dentro del directorio de imagenes/historias de Joomla. Luego, utilice el administrador de medios de Joomla para subir los videos/audios al folder correspondiente. Asegurese que en la configuración global de Joomla permita subir archivos diferente a imagenes.

Cuando este editando su contenido, utilice el nuevo botón AVR Media en la parte de abajo de la ventana del editor para insertar medios. Si desea embeber videos remotos almacenados en un proveedor de videos, simplemente pegue la URL de la página que muestra el video en el primer campo del dialogo que aparece al hacer clic en el botón, luego presione RETURN para iniciar el auto-reconocimiento. Si la URL es soportada, el resto de la forma será llenado de forma automática. El último campo presenta la etiqueta del medio la cual puede ser editada para añadir atributos personalizados. Para finalizar, presione el botón de Insertar y el contenido de este campo será insertado en la posición donde se encuentre actualmente el cursor en el editor.

Para insertar medios locales, use el mismo botón AVR Media, pero esta vez seleccione el tab de Medios Locales. Allí, usted puede seleccionar uno de los archivos de medios cargados previamente. Luego de selecionar el archivos de medios, el resto de los campos seran llenados de forma automática.

Para mayor información - especialmente antes de usar las características avanzadas -, por favor lea la documentación incluida. Allvideos Reloaded esta tratando de hacer las cosas fáciles para usted, sin embargo, embeber videos es un asunto complejo.
